# Portfolio

Este es mi portafolio personal.

Lo hice para tener un lugar donde ir subiendo proyectos, scripts y cosas que voy aprendiendo poco a poco.
No intenta ser algo espectacular ni lleno de efectos, solo algo claro y fácil de mantener.

---

## ¿Qué es este proyecto?

Es un sitio estático hecho con Astro donde muestro:

- proyectos personales
- scripts pequeños
- pruebas
- cosas en progreso

La idea es que el sitio crezca junto conmigo, sin tener que rehacerlo cada vez.

---

## Tecnologías

- Astro
- HTML y CSS
- JavaScript básico
- JSON local para manejar los proyectos
- GitHub Pages para publicarlo

No hay backend ni base de datos.

---

## ¿Cómo funciona?

Los proyectos están definidos en un archivo JSON (`projects.json`).
Astro lee ese archivo en el build y genera el HTML.

Si agrego o cambio un proyecto:

1. edito el JSON
2. hago commit
3. GitHub Pages se actualiza solo

---

## Deploy

El sitio se publica automáticamente usando GitHub Actions.

URL: https://jamrz.github.io/portfolio/

---

## Analítica

Uso Cloudflare Web Analytics solo para saber cuántas visitas tiene el sitio.
No usa cookies ni banners.
